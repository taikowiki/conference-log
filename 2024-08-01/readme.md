## 레이팅

- round(1127.5 * 상수) 를 기준으로 티어나누기
- 상수 별 기준
    - 13855 이상: 오메가
    - 11275 이상: 사파이어
    - 9020 이상: 루비
    - 6765 이상: 골드
    - 4510 이상: 실버
    - 2255 이상: 브론즈
    - 2254 이하: Pearl
- 각 티어에서 5개 세부 등급 나누기
- 족고수는 세부 티어 없음
